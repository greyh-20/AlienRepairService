Alien Repair Service

GitHub Repository

Game Description

In a distant galaxy, an alien engineer named Engi travels through space on a mission to find and repair abandoned space stations scattered across the cosmos.

Game Objectives

  • Avoid Asteroids: Colliding with asteroids reduces the number of lives.
  • Visit Space Stations: Each time you visit a station, you regain lives and progress in the game. If you visit all the stations, you win the game.

    • Controls

    • W: Accelerate the spaceship forward.
    • S: Move the spaceship backward.
    • A (Normal mode): Turn the spaceship left.
    • D (Normal mode): Turn the spaceship right.
    • A (SuperSpeed Mode): Move the spaceship left.
    • D (SuperSpeed Mode): Move the spaceship right.
    • F: Toggle SuperSpeed Mode on/off.
    • P: Pause or resume the game.
